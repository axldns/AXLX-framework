Packageaxl.xdef.interfaces
Interfacepublic interface ixDisplayContainer extends ixDisplay, ixDef
Implementors xCarousel, xSprite

Interface that allows to identify, parse and proceed valid XML children of config file to functional Display Object Containers Every implementer must provide function to execute when initial creation is complete, as well as listen to Event.ADDED on it's own display list on order to be able to execute onElementAdded method.
