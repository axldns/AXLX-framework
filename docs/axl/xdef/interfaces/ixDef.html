nameproperty 
name:String

Each element should be name-referenceable. This is the core functionality; to allow all elements within XML config file to be found, instantiated, added to displaylist, animated, removed from displaylist, referenced from other elements, changed, functions to be executed, it has to be legitimated by name property. Unlike naming proprty ID or anyhow else, this allows easy integration with non-framework elements
