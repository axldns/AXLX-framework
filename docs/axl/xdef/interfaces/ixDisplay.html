Packageaxl.xdef.interfaces
Interfacepublic interface ixDisplay extends ixDef
Implementors xBitmap, xText

Interface that allows to identify, parse and proceed valid XML children of config file to functional, displayable elements of the project. Every element must handle it's own addition to stage. Ideally execute animations defined in meta.addedToStage, respect resetOnAddedToStage parameter, only enabling it's other "ENTER_FRAME" related functions when on stage.
