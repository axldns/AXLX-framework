  appReomoteSPLITfilename : Array
Modyfies config file sub-directory. Array where first element is regular expression, second "replace" argument. If xRoot.appRemote is defined, final location of config file is going to be composed by executing String.replace on value of xRoot.fileName with regular expression of this property. New string is going to be concatenated with appRemote defined in xRoot class. Default [/(\w+?)(_./,"$1/$1$2/"] means filename VG_promo will produce VG/VG_PROMO, which will look for config in appRemote/VG/VG_promo/VG_promo.xml If xRoot.appRemote is not defined, this property is ignored.
