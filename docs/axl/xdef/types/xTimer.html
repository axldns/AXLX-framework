Packageaxl.xdef.types
Classpublic class xTimer
InheritancexTimer Counter Object
Implements ixDef

Non-displayable class that provides advanced timer functions. Instantiated from:<timer/> Extends axl.utils.Counter by providing XML interface to it. Additionaly, ancestor's callback can be defined as uncompiled portion of code - an arguments for xRoot.binCommand.
