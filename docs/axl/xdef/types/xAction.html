Packageaxl.xdef.types
Classpublic class xAction
InheritancexAction Object
Implements ixDef

Class that allows to execute at runtime XML defined funciton. Instantiated from <act/> node and can be referenced from registry. This class is non displayable (lightweight) object that can be placed in config file and executed only via direct or referenced execute call.

codeproperty 
public var code:String

Property containing uncompiled code for binCommand. code="stage.removeChildren()" The string is going to be parsed / code evaluated on execution, every execution. Confusion may occur as, unlike with other in-attrubute-code-executions, you probably don't want to prepend your config "code" attrubute value with dolar sign, unless you want to reference code containing variable to somewhere else.

repeatproperty 
public var repeat:Number

Number of times action is going to be executed on a single call. Parameter as any other argument can be referenced dynamically and changed on the fly. Useful for loops. Iteration index is available during loop as iteration property of this.

The default value is 0 - just one exec.
