Classpublic class xObject
InheritancexObject Object
Implements ixDef
Subclasses xScript

Lightweight class for loading data and providing an easy acces to its contents, without accessing internal assset manager. Instantiated from: <data/> Data parsing is available right after defining src attribute. Loaded data passes through known types check in order to instantiate valid ActionScript objects, according to Ldr.load Example: 
 <data src='audio.mp3' anything='$this.data.play()'/>
