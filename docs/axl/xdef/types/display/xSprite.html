Classpublic class xSprite
InheritancexSprite flash.display.Sprite
Implements ixDisplayContainer
Subclasses xButton, xForm, xMasked, xRoot, xScroll, xSwf, xVOD

Main DisplayObjectContainer class for XML defined objects. Can contain any children. Instantiated from: Overrides standard addChild, addChildAt, removeChild, removeChildAt methods in order to maintain meta-keyword-defined animations before removal or after addition children to display list. Self-listens for adding and removing from stage in order to maintain meta.addedToStage defined animation and to kill all existing animations when removed. Children of xml node will automatically become children of xSprite instance. Example: 
<div>	<img src='image1.png'/>	<img src='image2.png'/></div>

metaproperty 
meta:Object

Dynamic variables container. It's set up only once. Subsequent applying XML attributes or calling reset() will not have an effect.

  • "addedToStage" - animation(s) to execute when added to stage
  • "addChild" - animation to execute when added as a child
  • "removeChild" - animation to execute before removing from stage (delays removing from stage), instantiated and added to this instance
