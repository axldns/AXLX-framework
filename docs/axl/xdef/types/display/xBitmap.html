Packageaxl.xdef.types.display
Classpublic class xBitmap
InheritancexBitmap flash.display.Bitmap
Implements ixDisplay

Most basic object for displaying images. Since this class extends flash.display.Bitmap - it does not dispatch/receive mouse events. Instantiated from: Img nodes can not consist sub-nodes translatable to other display object (can't contain children). Allowed sub-nodes:

metaproperty 
meta:Object

Dynamic variables container. It's set up only once. Subsequent applying XML attributes or calling reset() will not have an effect.

  • "addedToStage" - animation(s) to execute when added to stage
  • "addChild" - animation to execute when added as a child
  • "removeChild" - animation to execute before removing from stage (delays removing from stage) instantiated and added to this instance
