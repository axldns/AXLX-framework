Packageaxl.xdef.types.display
Classpublic class xMasked
InheritancexMasked xSprite flash.display.Sprite

Class instantiated from <msk/> node, extends xSprite by adding three functionalites:
  • provides ready to use and already assigned, rectangular mask - controllable by properties visibleWidth, visibleHeight, visibleX, visibleY
  • provides controller property - an instance of BoundBox class, which allows to scroll, drag and move masked contents within masked area
  • exposes ability to make bi-directional connections with any other BoundBox controllers, e.g. members of xScroll (<scrollBar/>) instance
Instance auto-recognizes xScroll instance added to it's display list, places it outside of masked container and auto-connects scroll bar's controller to xMasked controller. Since method above is just a short-hand kept for backward ccompatibility, all other controllers should be connected via connectController function. Initial controller initial values can be overriden, default values as follows: ctrl.horizontal = true;ctrl.vertical = true;ctrl.horizontalBehavior = BoundBox.described; ctrl.verticalBehavior = BoundBox.described;ctrl.liveChanges = true; Values for bound and box of controller should never be changed.
